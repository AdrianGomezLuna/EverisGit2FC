# EverisGit2FC
Proyecto de formación - GIT 2

Enunciado:
1. Fork sobre el repositorio remoto que se indica a continuación:
2. Clonar el repositorio remoto en local.
3. Crear una nueva rama de desarrollo con el nombre devIniciales.
4. Establecer la rama de desarrollo como activa.
5. Realizar un cambio sobre el proyecto añadiendo un hola mundo con el nombre del/la desarrollador/a.
6. Añadir y commitear el cambio a la rama de desarrollo.
7. Integrar (merge) todo sobre la rama master.
8. Realizar push sobre el repositorio remoto en GitHub.
9. Realizar petición pull request para integración al repositorio original del primer punto.
